# Immobile Maze

Immobile Maze is a Python text-based adventure game.

## Installation

Python 3.8 is required to play this game. [Python 3.8](https://www.python.org/downloads/) to install Python.

## Run the Program

```bash
Python play.py
```
## Quit the Program

input quit or q when it's notated
